
# Pomodoro Timer (Notion-friendly)

- Enkele HTML file, klaar voor GitHub Pages.
- Exporteert CSV voor Notes en Tasks. Fallback modal aanwezig.
- LocalStorage voor log en dropdowns.
- Donkere UI met leesbaar contrast. Focus/break badge via CSS classes.

## Gebruik
Open `index.html` lokaal of via GitHub Pages. Druk `skip` om te testen.

## Host via GitHub Pages
1. Maak een nieuwe repo (public).
2. Upload `index.html` en `.nojekyll` naar de root.
3. Settings → Pages → Source: Deploy from a branch, Branch: main (root). Save.
4. Site: `https://<username>.github.io/<repo>/`.\
